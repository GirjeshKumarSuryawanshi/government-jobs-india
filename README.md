# Information Verification Process

Accuracy and transparency are core principles of GovtJobNow. Every recruitment article is prepared using information published by the respective recruiting authority.

Before publishing, our editorial process includes:

1. Reviewing the official recruitment notification.
2. Verifying important dates and application deadlines.
3. Checking eligibility criteria and educational qualifications.
4. Confirming vacancy details and reservation information.
5. Including direct links to official notifications and application portals.
6. Updating articles whenever official corrigenda or amendments are released.

Although every effort is made to maintain accuracy, candidates should always read the official notification before submitting an application.

# XML Sitemap

The XML Sitemap provides search engines with a structured list of discoverable pages.

Its goals include:

- Faster discovery of new content

- Better visibility of updated resources

# Robots.txt

The Robots.txt file provides crawl guidance for automated agents.

It is used to:

- Reference the XML Sitemap
- Guide crawler access
- Reduce unnecessary crawling of non-public resources
- Support efficient indexing

Robots.txt is reviewed periodically as the website evolves.
