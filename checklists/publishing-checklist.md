# Content Publishing Checklist

## Introduction

Publishing high-quality content requires more than writing an article.

Before a page goes live, it should be reviewed for accuracy, readability, technical quality, accessibility, and search readiness.

This checklist provides a repeatable workflow for publishing Government Job notifications and educational content.

---

# Phase 1 – Content Planning

Before writing:

- [ ] Primary topic selected.
- [ ] User search intent identified.
- [ ] Official notification collected.
- [ ] Required supporting documents available.
- [ ] Page type determined.
- [ ] Category selected.

---

# Phase 2 – Content Creation

Verify:

- [ ] Accurate title.
- [ ] Clear introduction.
- [ ] Organization name.
- [ ] Vacancy details.
- [ ] Eligibility criteria.
- [ ] Age limit.
- [ ] Salary information.
- [ ] Selection process.
- [ ] Application fee.
- [ ] Important dates.
- [ ] Required documents.
- [ ] Official notification.
- [ ] Official application link.

---

# Phase 3 – Readability

Review:

- [ ] Short paragraphs.
- [ ] Logical headings.
- [ ] Bullet lists used.
- [ ] Tables used where appropriate.
- [ ] Grammar checked.
- [ ] Spelling reviewed.
- [ ] Consistent terminology.

---

# Phase 4 – SEO

Review:

- [ ] Unique page title.
- [ ] Meta description.
- [ ] Clean URL.
- [ ] One H1 heading.
- [ ] Logical H2 hierarchy.
- [ ] Internal links added.
- [ ] External links verified.
- [ ] Canonical URL reviewed.

---

# Phase 5 – Structured Data

Review:

- [ ] Appropriate schema selected.
- [ ] JSON-LD valid.
- [ ] Required properties completed.
- [ ] Structured data matches visible content.

Examples:

- Organization
- Article
- BreadcrumbList
- FAQPage
- JobPosting (where appropriate)

---

# Phase 6 – Images

Review:

- [ ] Image optimized.
- [ ] Alt text written.
- [ ] Descriptive filename.
- [ ] Responsive display verified.

---

# Phase 7 – Mobile Experience

Review:

- [ ] Responsive layout.
- [ ] Tables readable.
- [ ] Images scale correctly.
- [ ] Buttons easy to tap.
- [ ] No horizontal scrolling.

---

# Phase 8 – Accessibility

Review:

- [ ] Heading hierarchy.
- [ ] Alternative text.
- [ ] Semantic HTML.
- [ ] Keyboard navigation.
- [ ] Color contrast.

---

# Phase 9 – AI Readiness (GEO)

Review:

- [ ] Topic clearly defined.
- [ ] Main entities identified.
- [ ] Context provided.
- [ ] Official sources referenced.
- [ ] Related resources linked.

---

# Phase 10 – Answer Readiness (AEO)

Review:

- [ ] Main question answered early.
- [ ] FAQ section included where appropriate.
- [ ] Tables used to simplify information.
- [ ] Headings reflect user questions.
- [ ] Answers are concise and complete.

---

# Phase 11 – Internal Linking

Verify links to:

- [ ] Parent category.
- [ ] Related recruitment.
- [ ] Admit card.
- [ ] Results.
- [ ] Syllabus.
- [ ] Previous papers.
- [ ] Official notification.

---

# Phase 12 – Technical Review

Confirm:

- [ ] Page loads successfully.
- [ ] HTTP 200 response.
- [ ] XML Sitemap updated (if applicable).
- [ ] robots.txt reviewed (if applicable).
- [ ] Structured data validated.
- [ ] Broken links checked.

---

# Phase 13 – Final Quality Assurance

Review:

- [ ] Dates verified.
- [ ] Numbers verified.
- [ ] Links verified.
- [ ] Formatting reviewed.
- [ ] Duplicate content avoided.
- [ ] Sensitive information reviewed.

---

# Phase 14 – Publish

After publishing:

- [ ] Verify page is live.
- [ ] Review formatting on desktop.
- [ ] Review formatting on mobile.
- [ ] Test important links.
- [ ] Confirm structured data remains valid.

---

# Phase 15 – Post-Publication

Within the next few days:

- [ ] Monitor for official updates.
- [ ] Correct any reported issues.
- [ ] Refresh information if the notification changes.
- [ ] Update related pages if necessary.

---

# Common Publishing Mistakes

Avoid:

- Publishing before proofreading.
- Missing official notification links.
- Outdated eligibility information.
- Incorrect dates.
- Missing metadata.
- Broken links.
- Invalid structured data.
- Duplicate pages.

---

# Government Jobs Specific Checklist

For every recruitment article:

- [ ] Notification number verified.
- [ ] Recruiting organization confirmed.
- [ ] Vacancy count reviewed.
- [ ] Reservation details reviewed (if applicable).
- [ ] Official PDF linked.
- [ ] Official apply link verified.
- [ ] Last date confirmed.
- [ ] Salary information reviewed.
- [ ] Selection process reviewed.

---

# Monthly Publishing Audit

Review:

- [ ] Expired job notifications.
- [ ] Expired application links.
- [ ] Broken internal links.
- [ ] Outdated FAQs.
- [ ] Missing schema.
- [ ] Updated official notifications.
- [ ] Content freshness.
